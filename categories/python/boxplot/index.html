박스플롯 그래프

박스플롯 그래프는 범주형 데이터 기준으로 수치형 데이터의 분포를 파악하는데 적합하다.

박스플롯을 보면 최솟값, 1분위, 중간값, 3분위 값, 최대값을 제공한다.
